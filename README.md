💰 Expense Tracker
📖 Overview

The Expense Tracker is a web application designed to help users manage their personal finances by tracking income and expenses in a simple and organized way. It allows users to record transactions, monitor their financial activities, and maintain a clear overview of their balance through an intuitive and user-friendly interface.

This project was developed to strengthen front-end web development skills by implementing responsive design, dynamic data handling, and interactive user interfaces using modern web technologies. It demonstrates practical concepts such as DOM manipulation, event handling, and real-time calculations.

✨ Features
Add income and expense transactions
Delete transactions
Real-time balance calculation
Separate income and expense summaries
Transaction history
Responsive design for desktop and mobile devices
User-friendly interface
Instant updates without page refresh
Clean and organized layout
Lightweight and fast performance
🛠 Technologies Used
HTML

HTML is used to create the structure of the application, including forms, transaction lists, buttons, and balance display.

CSS

CSS is used to design the interface with responsive layouts, modern styling, colors, typography, and spacing for a better user experience.

JavaScript

JavaScript powers the application by handling user input, calculating balances, updating transactions dynamically, and managing interactive features.

🚀 How It Works
Open the Expense Tracker application.
Enter the transaction title and amount.
Select whether the transaction is income or an expense.
Add the transaction to the list.
The application automatically updates:
Current Balance
Total Income
Total Expenses
Users can remove transactions whenever needed.
